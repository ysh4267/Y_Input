      동기화 (GitHub Gist)

      여러 PC에서 매크로를 자동으로 맞춥니다. 저장소를 만들 필요 없이 토큰만 넣으면 앱이 비공개(secret) gist를 자동 생성합니다. 같은 토큰을 쓰는 다른 PC도 그 gist를 자동으로 찾습니다.

      🔗 토큰 발급 페이지 열기 — gist 권한이 자동 체크됩니다. 맨 아래 Generate token 후 나온 ghp_…를 아래에 붙여넣으세요. (로그인은 브라우저에서)

      게임 오버레이

      현재 포커스된 게임 창 왼쪽 중앙에 활성화된 매크로 전부를 반투명 pill(2중 원 + 이름/루프)로 띄웁니다. 재생 중이 아니어도 표시되고, 재생되면 그 원이 채워집니다. 클릭은 게임으로 그대로 통과됩니다.
